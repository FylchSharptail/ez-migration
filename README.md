# EzGuild Migration Explorer

A single self-contained HTML page (no framework, no build step, no network) that ranks candidate Hero Wars guilds for migrating our 20-player squad.

One slider drives every guild at once; every number is intrinsic to that guild and varies row to row. A live in-page self-test asserts the correctness of the data shown, and a floating dock lets you change the squad-seat count from anywhere on the page.

**Live site:** https://fylchsharptail.github.io/ez-migration/

Source of the page lives in the guild tooling repo (hw_recovery/web). This repo only publishes the built artifact as `index.html`.
